Console Guide

How to join servers on Xbox One/PS4/Nintendo Switch

  • Navigate to "DNS settings" within the console's network settings.
  • Manually change the DNS. Set the primary DNS to "104.238.130.180" and the secondary DNS to "8.8.8.8".
  • Save the changes and restart the console.
  • Open Minecraft and join any featured server. If it worked, it will open to the server list.
  • Add a server. For the server address, enter "play.pgfmc.net". The port should be "19132".

If you're having trouble connecting, feel free to join the Discord.
